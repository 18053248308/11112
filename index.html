旅行保险赔付范围说明

医疗与意外伤害保障
✓
可赔付情形
以下情况符合保险责任范围，可申请理赔
✓

突发急性疾病

旅行期间首次发生的急性病，如急性肠胃炎、食物中毒、急性阑尾炎、突发高烧、严重感冒并发症等，需在保险期间内首次发病并接受治疗。

✓

意外伤害医疗

因意外事故导致的身体伤害，包括骨折、扭伤、擦伤、烧伤等。涵盖门诊治疗、住院费用、手术费用、处方药物及必要的医疗检查（X光、CT等）。

✓

紧急医疗运送与送返

因病情严重需要紧急医疗转运（如救护车、医疗专机）至最近合适医疗机构，或在治疗结束后送返常住地的合理费用。

✓

住院津贴

因意外或突发急性病在境外旅行期间住院治疗，按约定天数给付住院津贴，通常有最高赔偿天数限制（如累计不超过90天）。

✓

紧急牙科治疗

因意外事故导致的牙齿损伤所需的紧急治疗（如意外摔断牙齿），通常不包括常规牙科护理或原有牙齿问题的治疗。

✓

合理且必要的后续治疗

在旅行结束后一定期限内（如30天内），因旅行期间发生的保险事故所需的后续门诊复查或治疗费用。

✕
不可赔付情形（责任免除）
以下情况不在保险保障范围内，保险公司不承担赔偿责任
✕

既往病症与慢性病

投保前已存在的疾病、慢性病、遗传性疾病、先天性疾病或缺陷、艾滋病及相关病毒感染、性传播疾病、精神类疾病等导致的医疗费用。

✕

妊娠相关医疗

怀孕、分娩（含难产）、流产、堕胎、节育（含绝育）、产前产后检查及由以上原因引起的并发症。常规妊娠检查及与怀孕相关的任何医疗费用均不承保。

✕

美容整形与 elective 治疗

整容手术、美容治疗、矫形手术、洗牙洁齿、视力矫正、心理咨询、体检疗养、康复治疗、健康护理等非紧急医疗行为。

✕

高风险运动与违规行为

参与潜水、滑雪、攀岩、跳伞、蹦极、赛车、特技表演等高风险运动（除非特别附加承保）；酒后驾驶、无证驾驶、吸毒或服用管制药物期间发生的事故。

✕

故意行为与违法行为

投保人的故意行为；被保险人自伤、自杀（无民事行为能力人除外）、犯罪或拒捕；违背医嘱旅行；旅行目的为寻求或接受医疗治疗。

✕

战争、恐怖活动与核污染

战争、军事行动、暴动或武装叛乱期间；被保险人从事或参与恐怖主义活动、邪教组织活动；核爆炸、核辐射或核污染导致的伤害。

✕

常规预防性医疗

疫苗接种、常规体检、预防性治疗、慢性病维持用药、配镜（眼镜/隐形眼镜）、助听器、轮椅等辅助器具的购置费用。

✕

非指定医疗机构

在境外非正规医疗机构、美容诊所、康复中心等非保险人指定或认可的医疗机构发生的费用（紧急情况下的合理就医除外）。
